<h1> Shopism Backend </h1>

Create .env file and put MongoDB credential and app-signature for jwt token there

<hr>

### Start Production Server
```bash 
    #install pm2 i.e, process manager 
    $ > npm -g i pm2
    $ > npm i
    $ > npm runserver 
```
<hr>

### Testing Server
```bash
    #install nodemon first
    $ > npm -g i nodemon
    $ > npm i
    $ > npm test
```
